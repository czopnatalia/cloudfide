# Project Overview
This project implements a function called add_virtual_column designed to create a new pandas DataFrame that includes original data plus an additional column calculated based on specified mathematical operations.

# Instructions to run tests

1. You need to have pandas and pytest installed:  
*pip install pandas pytest*

2. To run the automated test suite, navigate to the project directory in your terminal and execute:   
*pytest "test_virtual_column 1.py"*
